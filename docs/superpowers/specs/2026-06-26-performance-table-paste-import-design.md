# 业绩年度总表粘贴导入设计

## 目标

在业绩项目库中支持从年度总表复制表格内容并批量生成业绩项目，降低手动逐条录入成本。

## 范围

- 业绩项目页新增“批量导入总表”按钮。
- 用户从 Excel/WPS/PDF 表格识别结果中复制多行文本，粘贴到文本框。
- 前端解析表头和行数据，生成预览。
- 用户确认后逐条调用现有新增业绩项目接口。

## 支持表头

按中文表头映射字段：

- 项目名称 / 项目 / 业绩名称 → `project_name`
- 甲方 / 客户 / 建设单位 / 发包方 → `client_name`
- 合同编号 / 合同号 → `contract_no`
- 合同金额 / 金额 → `contract_amount`
- 签订日期 / 合同签订日期 → `sign_date`
- 完成日期 / 验收日期 → `completion_date`
- 项目内容 / 供货范围 / 服务内容 → `project_scope`
- 年度 / 所属年度 → `year`
- 备注 → `remark`

## 不做

- 不解析 PDF 文件本身。
- 不自动关联合同证明文件。
- 不做后端批量接口。
- 不做复杂 OCR 表格修复。
