# 资质列表批量操作设计

## 目标

解决资质文件批量上传时选错分类的问题，并支持用户删除错分数据后按本地已整理分类重新上传。

## 功能范围

- 资质列表支持多选。
- 支持批量改分类。
- 支持批量删除。
- 批量删除采用源文件级语义：
  - 选中的资质若有关联 `file_id`，删除对应 `knowledge_files` 记录、该文件解析出的全部 `qualifications`、物理文件。
  - 选中的资质若无 `file_id`，仅删除该资质记录。
- 批量改分类同步更新：
  - 选中的 `qualifications.category`
  - 关联的 `knowledge_files.category`

## 安全规则

- 批量删除前必须二次确认。
- 确认文案明确说明将删除多少个源文件、多少条无源文件资质记录。
- 前端只全选当前页，不跨页隐式选择。
- 操作完成后清空选中项并刷新文件列表和资质列表。

## 不做

- 不做自动分类识别。
- 不做回收站。
- 不改数据库结构。
- 不改变单条删除原有行为。
